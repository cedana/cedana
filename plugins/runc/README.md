# Cedana Runc Plugin

Adds runc C/R support for Cedana.

## Support Matrix

| Feature | Supported | CLI                   |
| ------- | --------- | --------------------- |
| Dump    | Yes       | `cedana dump runc`    |
| Restore | Yes       | `cedana restore runc` |
| Run     | Yes       | `cedana run runc`     |
